## 📌 Project Overview

This project analyzes a healthcare appointment dataset to understand patient behavior, appointment trends, waiting times, hospital departments, and billing patterns.

The goal of this project is to clean the data, create useful features, answer business questions, and find insights that can help hospitals improve patient services and decision making.

The analysis was done using Python with Pandas, NumPy, and Matplotlib inside Jupyter Notebook.

---

## 🎯 Objectives

* Understand the healthcare dataset.
* Clean and prepare the data for analysis.
* Create new features to improve analysis.
* Explore patient appointment patterns.
* Answer business questions using data.
* Find useful insights for hospital management.

---

## 📊 Dataset Information

**Dataset Name:** Smart Healthcare Patient Appointment Dataset

**Source:** Kaggle

* Records: **10,000**
* Original Columns: **26**
* New Features Created: **4**
* Final Columns: **30**

### Feature Engineering

The following new columns were created during the project:

* Age Group
* Long Wait Flag
* Repeat Patient
* BMI Category

These features helped perform better analysis and answer more business questions.

## ❓ Business Questions Answered

This project answers several business questions, including:

* Which age group visits the hospital most frequently?
* Which BMI category is most common?
* What percentage of patients have no chronic disease?
* Which chronic diseases are most common?
* What percentage of patients have insurance?
* Which department receives the highest number of patients?
* Which appointment type is most preferred?
* Which weekday gets the highest number of appointments?
* Which month has the highest number of patients?
* Do patients with longer waiting times report lower satisfaction?
* Which department has the highest average waiting time?
* Do emergency patients wait less than normal patients?
* Do severe patients have higher medical bills?
* Which department handles the most emergency cases?
* Does severity level affect patient satisfaction?
* Is there a relationship between doctor experience and consultation fees?
* Does doctor experience affect patient satisfaction?
* Which department has the highest average doctor experience?
* Are emergency cases handled by more experienced doctors?
* Which department generates the highest revenue?
* How does insurance affect medical bills?
* How do medical bills vary across age groups?

## 💡 Key Insights

Some important insights from the analysis include:

* Patient visits vary across different age groups.
* Hospital departments receive different numbers of patients.
* Waiting time is not the same across departments.
* Emergency cases are handled differently than normal appointments.
* Insurance coverage affects patient billing.
* Doctor experience varies between departments.
* Some departments generate much higher revenue than others.
